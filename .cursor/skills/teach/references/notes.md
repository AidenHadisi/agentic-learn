# How to Write Notes

Notes are the session record for a section: what happened, what changed, and what a future agent should know before continuing.

## What to capture

- Key concepts covered and how they were explained
- Things the student struggled with or got confused by
- Questions the student asked and the answers given
- How the exercise went: their approach, mistakes, and what clicked
- Anything worth revisiting later

## Style

Write in clear, scannable markdown. Use headers, bullets, and code blocks when helpful. Keep it concise - these are notes, not a transcript. A future agent should be able to read them quickly and know what happened.
